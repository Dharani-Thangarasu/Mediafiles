# Chatbot Glossary

A reference guide to key terms and concepts related to the Document360 Chatbot.

---

## A

**Answer Confidence Score**
A percentage value (0–100%) that indicates how confident the AI is that a retrieved answer matches the user's query.

---

## B

**Bot Avatar**
The image or icon displayed in the chatbot widget to visually represent the chatbot.

**Bot Name**
The display name shown in the chat header that users see when interacting with the chatbot.

---

## C

**Confidence Threshold**
The minimum confidence score required for the chatbot to return an answer. Queries below this threshold trigger the fallback message.

**Conversation History**
A stored log of all past chatbot interactions, accessible from the analytics dashboard.

---

## E

**Escalation**
The process of transferring an unresolved chatbot conversation to a human support agent.

---

## F

**Fallback Message**
The response shown to users when the chatbot cannot find a matching answer.

---

## I

**Indexing**
The process of crawling and cataloging published articles so the chatbot can search them.

---

## N

**NLP (Natural Language Processing)**
The AI technology that enables the chatbot to understand and interpret user queries written in natural language.

---

## R

**Re-training**
Manually triggering the chatbot to re-index all published articles, typically done after bulk content updates.

**Resolved Rate**
The percentage of chatbot conversations in which the user's query was successfully answered.

---

## W

**Widget**
The chat bubble UI element embedded on the knowledge base portal or external website that users click to open the chatbot.

---

## Summary

This glossary will be updated as new features and terminology are added to the Document360 Chatbot.
