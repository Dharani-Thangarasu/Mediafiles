# Chatbot Changelog

A history of updates, improvements, and new features added to the Document360 Chatbot.

---

## Version 3.5.0 – March 2025

### New Features
- **Multi-language auto-detection**: The chatbot now automatically detects the user's language from their browser settings.
- **Conversation export**: Admins can export conversation history as PDF in addition to CSV.

### Improvements
- Improved NLP model accuracy for technical queries.
- Faster indexing after article updates (reduced from 5 minutes to under 1 minute).

### Bug Fixes
- Fixed an issue where the widget did not render correctly on iOS Safari 17.
- Resolved a rare race condition in the escalation webhook trigger.

---

## Version 3.4.0 – January 2025

### New Features
- **Chatbot Versioning**: Save and restore configuration snapshots.
- **Star Rating Feedback**: Added 5-star rating option alongside thumbs up/down.

### Improvements
- Analytics dashboard redesigned for improved readability.
- Escalation notification emails now include the full conversation transcript.

---

## Version 3.3.0 – October 2024

### New Features
- **Microsoft Teams Notification Integration**.
- **Custom CSS Support** for widget styling.

### Improvements
- Confidence score is now shown in the Analytics conversation detail view.
- Fallback message editor now supports rich text formatting.

---

## Version 3.2.0 – July 2024

### New Features
- **Chatbot API v1 released** – Query the chatbot programmatically.
- **User identification** support for logged-in portal visitors.

---

## Summary

The Document360 Chatbot is continuously improved with each release. Check this page regularly for the latest updates.
