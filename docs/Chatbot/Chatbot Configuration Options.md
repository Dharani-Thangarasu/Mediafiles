# Chatbot Configuration Options

Document360 provides a range of configuration options to tailor the chatbot experience to your brand and audience.

---

## General Settings

| Setting             | Description                                              |
|---------------------|----------------------------------------------------------|
| **Bot Name**        | The display name shown to users in the chat widget.      |
| **Avatar**          | Upload a custom icon or image for the bot.               |
| **Language**        | Set the default language for bot responses.              |
| **Theme Color**     | Match the chatbot widget color to your brand palette.    |

---

## Conversation Settings

- **Welcome Message** – The first message shown when a user opens the chat.
- **Fallback Message** – Displayed when no matching answer is found.
- **Escalation Message** – Shown when the bot suggests contacting human support.

---

## Answer Sources

You can restrict chatbot answers to specific:
- **Categories**
- **Article tags**
- **Language versions**

---

## Confidence Threshold

Set a minimum confidence score (0–100%) below which the bot will show a fallback message instead of a low-quality answer.

---

## Summary

Fine-tuning the chatbot configuration ensures users receive accurate, branded, and helpful responses every time they interact with it.
