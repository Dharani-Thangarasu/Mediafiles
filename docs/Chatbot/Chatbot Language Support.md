# Chatbot Language Support

The Document360 Chatbot supports multiple languages, enabling you to serve a global audience in their preferred language.

---

## Supported Languages

The chatbot can deliver answers in any language that your knowledge base is configured for, including but not limited to:

- English
- French
- German
- Spanish
- Portuguese
- Japanese
- Chinese (Simplified)
- Arabic
- Dutch
- Italian

---

## How Language Detection Works

1. The chatbot detects the language of the user's query automatically.
2. It searches for matching content in the corresponding language version of the knowledge base.
3. If no translation is available, the bot falls back to the default language.

---

## Setting the Default Language

1. Go to **Settings → Chatbot → Language**.
2. Select the **Default Language** from the dropdown.
3. Save your changes.

---

## Enabling Multi-Language Support

1. Ensure your knowledge base has articles published in the target language.
2. Enable the language version in **Settings → Languages**.
3. The chatbot will automatically serve answers in that language.

---

## Fallback Behavior

When the chatbot cannot find an answer in the user's detected language, it will:
1. Attempt to return an answer from the default language.
2. Show the configured **Fallback Message** if no match is found.

---

## Summary

Multi-language chatbot support helps you provide a consistent, localized self-service experience for users around the world.
