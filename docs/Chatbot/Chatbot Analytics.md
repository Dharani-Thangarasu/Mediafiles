# Chatbot Analytics

Monitor how your chatbot is performing and identify areas for improvement using the built-in analytics dashboard.

---

## Accessing Analytics

Navigate to **Analytics → Chatbot** in your Document360 project to view performance metrics.

---

## Key Metrics

| Metric                     | Description                                                |
|----------------------------|------------------------------------------------------------|
| **Total Conversations**    | Number of chat sessions initiated.                         |
| **Questions Asked**        | Total number of user queries submitted.                    |
| **Resolved Rate**          | Percentage of questions answered successfully.             |
| **Escalation Rate**        | Percentage of conversations escalated to human support.    |
| **Top Questions**          | Most frequently asked questions.                           |
| **Unanswered Questions**   | Queries where the bot could not find a match.              |

---

## Filters

Refine your analytics view by:
- **Date range** – Today, Last 7 days, Last 30 days, Custom.
- **Language** – Filter by knowledge base language.
- **Category** – Focus on a specific documentation section.

---

## Exporting Data

Click **Export** to download analytics data as a CSV file for offline reporting or sharing with your team.

---

## Using Analytics to Improve the Bot

1. Review **Unanswered Questions** and create new articles to fill gaps.
2. Update existing articles that are failing to satisfy user queries.
3. Adjust the **Confidence Threshold** based on resolved rate trends.

---

## Summary

Chatbot analytics provide actionable insights that help you continuously refine your documentation and bot performance.
