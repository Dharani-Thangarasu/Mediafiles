# Chatbot Localization

Translate and localize chatbot messages, templates, and widget labels to serve users in their native language.

---

## What Is Chatbot Localization?

Localization goes beyond language detection. It involves translating all chatbot-facing text—such as the welcome message, fallback message, and widget labels—into each supported language.

---

## Supported Localization Elements

| Element                  | Description                                           |
|--------------------------|-------------------------------------------------------|
| **Welcome Message**      | The greeting shown when the chat opens.               |
| **Fallback Message**     | Shown when no answer is found.                        |
| **Escalation Message**   | Shown when routing to human support.                  |
| **Feedback Prompt**      | "Was this helpful?" text and button labels.           |
| **Input Placeholder**    | Text inside the query input field.                    |
| **Widget Tooltip**       | Text shown on hover over the chat bubble.             |

---

## Adding Translations

1. Go to **Settings → Chatbot → Localization**.
2. Select a language from the language dropdown.
3. Enter translated text for each localization element.
4. Click **Save**.

---

## Fallback Language

If a user queries in a language that has no translated messages configured, the chatbot falls back to the **Default Language** set in general settings.

---

## RTL Language Support

For right-to-left languages such as Arabic and Hebrew, the chatbot widget layout automatically mirrors to ensure a natural reading experience.

---

## Tips for Good Translations

- Use native speakers for translations rather than machine translation for critical messages.
- Keep messages concise — chatbot messages should be 1–2 sentences maximum.
- Match the tone of your brand across all languages.

---

## Summary

Localizing your chatbot ensures that users across all regions receive a consistent, native-language experience that builds trust and improves satisfaction.
