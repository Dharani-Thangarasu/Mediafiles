# Chatbot Data Export

Export chatbot conversation data and analytics for offline reporting, compliance, or integration with external analytics platforms.

---

## What Can You Export?

| Data Type                  | Format         | Description                                               |
|----------------------------|----------------|-----------------------------------------------------------|
| **Conversation History**   | CSV, PDF       | Full logs of all chat sessions and responses.             |
| **Analytics Summary**      | CSV            | Aggregated metrics (resolved rate, top queries, etc.).    |
| **Unanswered Questions**   | CSV            | List of queries the bot could not answer.                 |
| **Feedback Data**          | CSV            | User ratings and comments per conversation.               |

---

## Exporting Conversation History

1. Navigate to **Analytics → Chatbot → Conversations**.
2. Apply date range and filters as needed.
3. Click **Export → CSV** or **Export → PDF**.
4. The file downloads immediately or is emailed to you for large exports.

---

## Exporting Analytics Data

1. Navigate to **Analytics → Chatbot → Overview**.
2. Select your desired date range.
3. Click **Export CSV**.

---

## Scheduled Exports (Enterprise)

Enterprise plan users can set up **Scheduled Exports** to automatically receive data exports by email:

1. Go to **Settings → Chatbot → Scheduled Exports**.
2. Choose frequency: **Daily**, **Weekly**, or **Monthly**.
3. Enter recipient email addresses.
4. Select data types to include.
5. Save.

---

## Using Exported Data

Common uses for exported chatbot data include:
- Building custom dashboards in Power BI or Tableau.
- Sharing performance reports with stakeholders.
- Feeding query data into external NLP training pipelines.
- Compliance and audit reporting.

---

## Summary

Flexible data export options give you full ownership of your chatbot data, enabling reporting and integration workflows outside of Document360.
