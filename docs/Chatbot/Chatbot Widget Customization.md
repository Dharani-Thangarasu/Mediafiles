# Chatbot Widget Customization

Customize the appearance and behavior of the chatbot widget to align with your knowledge base's look and feel.

---

## Widget Appearance

### Position
Choose where the widget appears on the page:
- **Bottom-right** (default)
- **Bottom-left**

### Colors
- **Primary Color** – Background color of the chat header and send button.
- **Text Color** – Color of the bot's message text.
- **Background Color** – The chat window background.

### Widget Icon
Upload a custom icon (SVG or PNG, max 64×64 px) to replace the default chat bubble.

---

## Widget Behavior

| Option                   | Description                                              |
|--------------------------|----------------------------------------------------------|
| **Auto-open**            | Automatically open the chat window after a set delay.    |
| **Mobile Responsive**    | Ensure the widget adjusts correctly on small screens.    |
| **Show Article Links**   | Display links to full articles alongside answers.        |
| **Minimize on Load**     | Start the widget in minimized state.                     |

---

## Custom CSS

Advanced users can inject custom CSS to further style the widget beyond the built-in options.

---

## Preview

Use the **Live Preview** panel in settings to see how the widget looks before publishing changes.

---

## Summary

Widget customization lets you create a seamless, on-brand chat experience for your users without any coding required.
