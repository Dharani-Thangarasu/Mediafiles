# Zapier Integration

Connect Document360 with Zapier to automate documentation workflows by linking to thousands of third-party applications.

---

## Overview

The Zapier integration enables teams to build no-code automation workflows (Zaps) that trigger actions in Document360 or respond to Document360 events within thousands of connected apps.

---

## Prerequisites

- A Zapier account (Starter plan or above recommended)
- Document360 project admin access
- Document360 API key

---

## Setup Steps

1. Navigate to **Settings → Integrations → Zapier** in Document360.
2. Copy your Document360 **API key**.
3. Log in to Zapier and create a new Zap.
4. Search for and select **Document360** as the trigger or action app.
5. Authenticate using your Document360 API key.
6. Configure the Zap trigger and action steps as needed.
7. Turn on the Zap.

---

## Features

- **Trigger on Events** – Start Zaps when articles are created, updated, or published.
- **Create Articles** – Automatically create Document360 articles from other app events.
- **Multi-Step Zaps** – Chain multiple actions across apps in a single workflow.
- **5,000+ App Connections** – Integrate Document360 with virtually any tool via Zapier.

---

## Use Cases

- Automatically create a Document360 article draft when a Google Form is submitted.
- Notify a Slack channel when a new article is published using a Zap.
- Add a Trello card for every new Document360 draft article that needs review.

---

## Best Practices

- Use Zapier filters to ensure automations only trigger for relevant events.
- Test Zaps thoroughly with sample data before enabling them in production.
- Monitor Zap task history to detect and fix failed automation runs.

---

## Summary

The Zapier integration makes it easy to connect Document360 to the rest of your tool stack, enabling powerful no-code automation without requiring developer resources.
