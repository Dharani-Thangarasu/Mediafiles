# Mixpanel Integration

Use the Mixpanel integration with Document360 to track user interactions and analyze documentation engagement with event-based analytics.

---

## Overview

Mixpanel is a powerful product analytics platform that lets you track specific user actions. When integrated with Document360, you can build detailed funnels and retention analyses for your knowledge base.

---

## Prerequisites

- A Mixpanel account
- Your Mixpanel **Project Token**
- Document360 project admin access

---

## Setup Steps

1. Log in to Document360 and go to **Settings → Integrations → Mixpanel**.
2. Toggle the **Enable** switch.
3. Enter your **Mixpanel Project Token**.
4. Click **Save**.

Mixpanel will begin capturing events from your documentation site.

---

## Events Tracked

| Event Name | Description |
|---|---|
| `page_view` | User viewed an article |
| `article_search` | User performed a search |
| `feedback_submitted` | User submitted article feedback |
| `link_clicked` | User clicked an internal/external link |
| `category_navigated` | User browsed a category |

---

## Analyzing Data in Mixpanel

1. Log in to your Mixpanel project.
2. Navigate to **Events** to see raw event data.
3. Use **Funnels** to track conversion paths through your documentation.
4. Use **Retention** to see how often users return to the knowledge base.
5. Build **Dashboards** to monitor key documentation KPIs.

---

## Best Practices

- Define key documentation metrics (e.g., search-to-article conversion rate).
- Use Mixpanel cohorts to segment users by documentation behavior.
- Monitor search queries that return no results to identify content gaps.

---

## Summary

The Mixpanel integration gives you granular, event-level insight into how users navigate and interact with your documentation, helping you make data-driven improvements.
