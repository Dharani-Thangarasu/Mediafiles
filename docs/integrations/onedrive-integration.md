# OneDrive Integration

Connect Document360 with Microsoft OneDrive to import, manage, and embed documentation files from your cloud storage.

---

## Overview

The OneDrive integration allows teams to link OneDrive and SharePoint document libraries to Document360, enabling seamless file import and asset management within the knowledge base.

---

## Prerequisites

- A Microsoft 365 account with OneDrive access
- Document360 project admin access
- Microsoft OAuth credentials (tenant ID, client ID, client secret)

---

## Setup Steps

1. Navigate to **Settings → Integrations → OneDrive** in Document360.
2. Click **Connect with Microsoft OneDrive**.
3. Authorize Document360 using your Microsoft 365 account.
4. Select OneDrive folders or SharePoint libraries to link.
5. Configure import and sync settings.
6. Click **Save**.

---

## Features

- **File Import** – Import Word, Excel, PowerPoint, and PDF files from OneDrive.
- **Folder Sync** – Keep documentation in sync with OneDrive folder updates.
- **SharePoint Libraries** – Access SharePoint document libraries alongside personal OneDrive files.
- **Media Library Integration** – Use OneDrive images in the Document360 media library.

---

## Use Cases

- Import existing Word documents from OneDrive as Document360 articles.
- Sync SharePoint document libraries with a documentation knowledge base.
- Use OneDrive as a staging area for content before publishing to Document360.

---

## Best Practices

- Organize OneDrive folders by documentation category for easy mapping.
- Use SharePoint permissions to control which files are accessible to Document360.
- Review imported documents for formatting issues after conversion.

---

## Summary

The OneDrive integration connects Microsoft cloud storage with Document360, enabling teams already invested in Microsoft 365 to leverage their existing files for documentation.
