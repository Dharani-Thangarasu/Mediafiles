# Asana Integration

Integrate Document360 with Asana to manage documentation tasks alongside your team's broader project work.

---

## Overview

The Asana integration helps documentation teams track article creation, review, and publishing tasks within Asana projects, keeping documentation aligned with product timelines.

---

## Prerequisites

- An Asana workspace with project access
- Document360 project admin access
- Asana personal access token or OAuth credentials

---

## Setup Steps

1. Navigate to **Settings → Integrations → Asana** in Document360.
2. Click **Connect with Asana**.
3. Authorize Document360 to access your Asana workspace.
4. Select the Asana project(s) to link.
5. Configure task creation and status sync rules.
6. Click **Save**.

---

## Features

- **Task Linking** – Attach Document360 articles to Asana tasks.
- **Auto Task Creation** – Generate Asana tasks automatically when new articles are created.
- **Status Updates** – Reflect article publish status changes as Asana task completions.
- **Due Date Sync** – Align article deadlines with Asana task due dates.

---

## Use Cases

- Track documentation deliverables in the same Asana project as product features.
- Assign article review tasks to team members through Asana.
- Use Asana timelines to plan documentation releases alongside product launches.

---

## Best Practices

- Create a dedicated Asana project for documentation to separate it from other work.
- Use Asana custom fields to capture article metadata such as category and priority.
- Regularly review completed tasks to identify documentation gaps.

---

## Summary

The Asana integration makes documentation a first-class citizen in your project management workflows, ensuring content creation is planned and tracked with the same rigor as product development.
