# Tableau Integration

Connect Document360 with Tableau to embed interactive data visualizations and dashboards within documentation articles.

---

## Overview

The Tableau integration allows teams to embed live Tableau dashboards and visualizations directly into Document360 articles, enabling data-driven documentation that stays current as underlying data changes.

---

## Prerequisites

- A Tableau Online or Tableau Server account
- Document360 project admin access
- Tableau embed URL or JavaScript API access

---

## Setup Steps

1. Navigate to **Settings → Integrations → Tableau** in Document360.
2. Enable Tableau embed support in the article editor settings.
3. In Tableau, generate a shareable embed link for the desired view.
4. Paste the Tableau embed URL into a Document360 article using the embed widget.
5. Configure Tableau site-wide embedding permissions as needed.
6. Publish the article.

---

## Features

- **Interactive Embeds** – Embed live Tableau dashboards that users can interact with.
- **Filtered Views** – Embed specific filtered views of dashboards for contextual relevance.
- **Auto-Refresh** – Dashboards refresh automatically when underlying data is updated.
- **Access Control** – Respect Tableau permissions to ensure secure data access.

---

## Use Cases

- Embed live metrics dashboards in executive documentation or monthly reports.
- Include interactive data visualizations in product analytics documentation.
- Display real-time operational metrics in runbooks or incident response docs.

---

## Best Practices

- Use Tableau's "Hide Toolbar" embed option for cleaner documentation presentation.
- Ensure embedded views are accessible to the intended Document360 audience.
- Optimize Tableau view performance before embedding to avoid slow load times.

---

## Summary

The Tableau integration brings interactive, data-driven insights directly into documentation, transforming static articles into living, data-enriched resources.
