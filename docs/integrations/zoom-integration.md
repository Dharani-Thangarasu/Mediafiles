# Zoom Integration

Connect Document360 with Zoom to attach meeting recordings, webinar transcripts, and video walkthroughs to documentation articles.

---

## Overview

The Zoom integration allows teams to enrich documentation with video content from Zoom meetings and webinars, making knowledge bases more engaging and accessible.

---

## Prerequisites

- A Zoom account with recording access
- Document360 project admin access
- Zoom OAuth credentials (client ID and secret)

---

## Setup Steps

1. Navigate to **Settings → Integrations → Zoom** in Document360.
2. Click **Connect with Zoom**.
3. Authorize Document360 to access your Zoom account.
4. Browse and select recordings to attach to articles.
5. Configure auto-import rules for new recordings.
6. Click **Save**.

---

## Features

- **Recording Embeds** – Embed Zoom meeting and webinar recordings in documentation.
- **Transcript Import** – Automatically import Zoom transcripts as searchable article content.
- **Auto-Import** – Automatically add new recordings to a specified documentation section.
- **Shareable Links** – Generate password-protected links for sensitive recordings.

---

## Use Cases

- Embed product walkthrough recordings in onboarding documentation.
- Import webinar transcripts as searchable knowledge base articles.
- Link training session recordings to employee handbook pages.

---

## Best Practices

- Enable Zoom cloud recording to make recordings available for import.
- Review transcripts for accuracy before publishing them as documentation.
- Organize recordings into Zoom folders that map to documentation categories.

---

## Summary

The Zoom integration turns meeting and webinar recordings into valuable documentation assets, making it easy to share knowledge through both video and text.
