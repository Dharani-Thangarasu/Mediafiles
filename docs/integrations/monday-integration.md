# Monday.com Integration

Connect Document360 with Monday.com to manage documentation workflows within your team's work operating system.

---

## Overview

The Monday.com integration allows teams to track documentation tasks, deadlines, and publishing milestones directly on Monday.com boards, keeping docs aligned with broader organizational workflows.

---

## Prerequisites

- A Monday.com account with board admin permissions
- Document360 project admin access
- Monday.com API key

---

## Setup Steps

1. Navigate to **Settings → Integrations → Monday.com** in Document360.
2. Enter your Monday.com **API key**.
3. Select the board(s) to connect.
4. Map Document360 article states to Monday.com column statuses.
5. Configure automation triggers.
6. Click **Save**.

---

## Features

- **Board Integration** – Sync article tasks to Monday.com boards automatically.
- **Status Columns** – Map documentation stages (Draft, Review, Published) to Monday.com statuses.
- **Dependency Tracking** – Link documentation tasks to related Monday.com items.
- **Notification Automations** – Trigger Monday.com automations based on documentation events.

---

## Use Cases

- Plan content calendars using Monday.com's timeline and calendar views.
- Track article ownership and deadlines across the documentation team.
- Link feature documentation tasks to product development items on the same board.

---

## Best Practices

- Use Monday.com groups to organize articles by product area or release version.
- Set up Monday.com automations to notify writers when review is pending.
- Review board dashboards weekly to spot bottlenecks in the documentation pipeline.

---

## Summary

The Monday.com integration brings documentation management into your team's central work platform, enabling better planning, visibility, and accountability across the content lifecycle.
