# Figma Integration

Connect Document360 with Figma to embed design assets directly into your documentation articles.

---

## Overview

The Figma integration allows technical writers and product teams to embed live Figma frames, prototypes, and design assets into Document360 articles, keeping documentation visually accurate and up-to-date.

---

## Prerequisites

- A Figma account with file access permissions
- Document360 project admin access
- Figma Personal Access Token

---

## Setup Steps

1. Navigate to **Settings → Integrations → Figma** in Document360.
2. Enter your Figma **Personal Access Token**.
3. Authorize Document360 to access your Figma files.
4. Use the Figma embed widget within article editors to link frames.
5. Publish your article with the embedded Figma content.
6. Click **Save**.

---

## Features

- **Live Embeds** – Embed Figma frames that update automatically when the design changes.
- **Interactive Prototypes** – Include clickable Figma prototypes within documentation.
- **Asset Library** – Browse and insert Figma components directly from the article editor.
- **Version Linking** – Link specific Figma file versions to documentation releases.

---

## Use Cases

- Embed UI mockups directly into feature documentation for clarity.
- Use live Figma prototypes in onboarding guides to demonstrate workflows.
- Link design specifications to corresponding developer documentation.

---

## Best Practices

- Use Figma's share settings to ensure embedded frames are publicly accessible.
- Organize Figma files by product area to make linking to documentation easier.
- Review embedded frames when designs are updated to ensure documentation accuracy.

---

## Summary

The Figma integration enriches documentation with live, interactive design assets, ensuring that written content and visual design always tell the same story.
