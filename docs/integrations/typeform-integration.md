# Typeform Integration

Connect Document360 with Typeform to embed interactive forms and surveys within documentation articles.

---

## Overview

The Typeform integration allows teams to embed Typeform surveys, quizzes, and feedback forms directly into Document360 articles, collecting structured feedback from readers without redirecting them away from the knowledge base.

---

## Prerequisites

- A Typeform account with form creation access
- Document360 project admin access
- Typeform embed code or API access

---

## Setup Steps

1. Navigate to **Settings → Integrations → Typeform** in Document360.
2. Enter your Typeform **API token** or connect via OAuth.
3. Enable Typeform embed support in the article editor.
4. Use the embed widget to paste Typeform form URLs into articles.
5. Configure response webhook forwarding if needed.
6. Click **Save**.

---

## Features

- **Form Embeds** – Embed Typeform surveys directly within documentation articles.
- **Quiz Integration** – Add knowledge-check quizzes to training documentation.
- **Feedback Collection** – Collect structured reader feedback using conversational forms.
- **Response Webhooks** – Forward Typeform responses to other tools via webhooks.

---

## Use Cases

- Add a satisfaction survey at the end of troubleshooting articles.
- Embed a product feedback form within release notes documentation.
- Create interactive quizzes in onboarding documentation to assess comprehension.

---

## Best Practices

- Keep embedded forms short to avoid disrupting the reading experience.
- Use Typeform's logic jumps to personalize form experiences based on reader input.
- Review form responses regularly and update documentation based on feedback trends.

---

## Summary

The Typeform integration enriches documentation with interactive forms and surveys, making it easy to collect valuable reader feedback directly within the knowledge base.
