# Dropbox Integration

Connect Document360 with Dropbox to import, sync, and manage documentation assets stored in your Dropbox account.

---

## Overview

The Dropbox integration allows teams to link Dropbox folders to Document360, making it easy to import files and media assets directly into the knowledge base without manual uploads.

---

## Prerequisites

- A Dropbox account with folder access
- Document360 project admin access
- Dropbox API app credentials

---

## Setup Steps

1. Navigate to **Settings → Integrations → Dropbox** in Document360.
2. Click **Connect with Dropbox**.
3. Authorize Document360 to access your Dropbox account.
4. Select folders to sync with documentation categories.
5. Configure file import and update rules.
6. Click **Save**.

---

## Features

- **File Import** – Import documents and images from Dropbox directly into articles.
- **Folder Sync** – Automatically sync Dropbox folder changes to documentation.
- **Media Library** – Access Dropbox files from the Document360 media library.
- **Version Tracking** – Maintain Dropbox file version history within documentation.

---

## Use Cases

- Maintain a shared Dropbox folder for documentation drafts and import to Document360 on approval.
- Use Dropbox as a central asset repository for images used across documentation.
- Share Dropbox Paper documents with the Document360 team for collaborative editing.

---

## Best Practices

- Organize Dropbox folders to mirror Document360's category structure.
- Use Dropbox's file request feature to collect content from contributors.
- Enable Dropbox notifications to be alerted when shared folders are updated.

---

## Summary

The Dropbox integration simplifies asset management for documentation teams by connecting Document360 with a widely used cloud storage platform.
