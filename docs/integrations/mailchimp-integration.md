# Mailchimp Integration

Connect Document360 with Mailchimp to manage documentation newsletter subscriptions and email campaigns.

---

## Overview

The Mailchimp integration allows teams to grow and manage a subscriber list for documentation updates, send newsletters when new content is published, and track audience engagement.

---

## Prerequisites

- A Mailchimp account with list/audience admin access
- Document360 project admin access
- Mailchimp API key

---

## Setup Steps

1. Navigate to **Settings → Integrations → Mailchimp** in Document360.
2. Enter your Mailchimp **API key**.
3. Select the Mailchimp audience to sync with Document360 subscribers.
4. Configure subscription triggers (e.g., when a reader subscribes to updates).
5. Set up automation campaigns for new article notifications.
6. Click **Save**.

---

## Features

- **Subscriber Sync** – Automatically add Document360 subscribers to a Mailchimp audience.
- **Campaign Automation** – Trigger Mailchimp campaigns when new documentation is published.
- **Segmentation** – Segment subscribers by documentation category or reading behavior.
- **Engagement Analytics** – Track email open rates and clicks within Mailchimp.

---

## Use Cases

- Send a weekly newsletter summarizing newly published knowledge base articles.
- Segment subscribers by product interest and send targeted documentation updates.
- Automate a welcome email series for new knowledge base subscribers.

---

## Best Practices

- Use double opt-in for documentation newsletter subscriptions to ensure list quality.
- Segment audiences based on documentation categories they've subscribed to.
- Review campaign analytics regularly to optimize email content and send frequency.

---

## Summary

The Mailchimp integration transforms documentation subscribers into an engaged audience, enabling teams to communicate content updates through professional email campaigns.
