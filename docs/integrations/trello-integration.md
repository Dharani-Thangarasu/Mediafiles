# Trello Integration

Connect Document360 with Trello to link documentation tasks to boards and streamline content creation workflows.

---

## Overview

The Trello integration enables teams to manage documentation tasks on Trello boards while keeping the knowledge base updated, bridging project management with content creation.

---

## Prerequisites

- A Trello account with board access
- Document360 project admin access
- Trello API key and token

---

## Setup Steps

1. Navigate to **Settings → Integrations → Trello** in Document360.
2. Click **Connect with Trello**.
3. Authorize Document360 to access your Trello account.
4. Select the Trello board and lists to link.
5. Configure card-to-article mapping and automation rules.
6. Click **Save**.

---

## Features

- **Card Linking** – Attach Document360 articles to Trello cards for quick access.
- **Auto Card Creation** – Automatically create Trello cards when new articles are drafted.
- **Status Sync** – Update Trello card statuses when articles are published or archived.
- **Board Notifications** – Receive Trello notifications for documentation milestones.

---

## Use Cases

- Use Trello cards to plan and track new articles in a content calendar.
- Link release notes in Document360 to corresponding Trello release cards.
- Manage the documentation review process entirely within Trello.

---

## Best Practices

- Use dedicated Trello lists for each documentation stage (Draft, Review, Published).
- Add checklists to Trello cards to track individual article progress.
- Archive completed cards regularly to keep the board clean.

---

## Summary

The Trello integration brings visual project management to your documentation workflow, making it easy to plan, track, and publish content efficiently.
